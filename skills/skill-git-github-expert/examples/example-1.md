# Example 1

## Solicitud

"Revisa mi rama, separa los cambios si están mezclados y prepara un commit y PR con Conventional Commits".

## Comportamiento esperado

- inspeccionar `git status`, ramas y diffs;
- detectar mezcla de concerns;
- proponer o ejecutar el commit más pequeño viable;
- redactar un mensaje de PR con contexto, riesgos y verificación.
