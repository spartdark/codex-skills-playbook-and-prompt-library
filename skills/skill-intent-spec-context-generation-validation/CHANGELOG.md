# Changelog

## [0.2.0] - 2026-03-29

### Added

- version publica inicial de la skill;
- framework ISCGV con seis gates obligatorios;
- guia de prompt para producto AI-first y validacion;
- checklist de revision y ejemplo de activacion.

### Changed

- evolucion de ISCGV a ISCGV++ con arquitectura multi-agente;
- incorporacion de Audit Agent obligatorio despues de cada fase;
- incorporacion de scoring por claridad, completitud, riesgo y validabilidad;
- regla explicita de bloqueo cuando cualquier score sea menor a 3.
