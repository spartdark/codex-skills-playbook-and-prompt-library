# Example 1

## Solicitud

"Tengo una idea para un copiloto de ventas con IA, pero no quiero construir nada hasta saber si resuelve un problema real, que contexto necesita, como validarlo y donde deberiamos bloquear el avance si la idea es debil".

## Comportamiento esperado

- aterrizar el problema, usuario, urgencia y JTBD;
- generar un PRD con MVP experimental;
- definir estrategia de contexto, memoria y RAG;
- proponer guardrails, evals y criterios de falsificacion;
- auditar cada fase y asignar scores;
- cerrar con una recomendacion clara sobre avanzar, iterar o no construir.
