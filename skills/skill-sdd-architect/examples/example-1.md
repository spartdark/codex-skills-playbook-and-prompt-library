# Example 1

## Solicitud

"Quiero convertir una idea de feature en un PRD, un plan técnico y una lista de tareas para implementar en dos semanas".

## Comportamiento esperado

- aclarar el problema y el usuario objetivo;
- definir alcance y no alcance;
- escribir criterios de aceptación medibles;
- dividir la implementación por fases con riesgos y dependencias.
