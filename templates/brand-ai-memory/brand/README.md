# Brand Memory

AI-ready memory for this brand identity.

## Agent Load Order

1. Read this file.
2. Read `brand-memory.json`.
3. Read `brand-brief.md` when strategy, audience, tone, or positioning matters.
4. Read `production-checklist.md` for Cricut, vinyl, print, packaging, or signage.
5. Read relevant files in `decisions/` before changing approved direction.
6. Use assets in `assets/` as source references.

## Rules

- Treat `brand-memory.json` as the source of truth.
- Do not invent new colors, fonts, logo styles, or production rules unless clearly marked as a proposal.
- Preserve approved decisions and record new decisions in `decisions/`.
- Use prompts in `prompts/` as reusable recipes, not as fixed truth.
- For Cricut or vinyl, prefer flat vector SVG exports with outlined text and expanded strokes.
- Report validation performed before considering work done.

## Done For New Brand Assets

- Uses approved colors, typography, logo rules, and tone.
- References source assets used.
- Adds or updates a decision if brand rules changed.
- Updates `brand-memory.json` only when the learning is reusable.
- Records production validation when relevant.
