# Production Checklist

## General

- [ ] Source asset exists in `assets/logo/`.
- [ ] Exported asset name is stable.
- [ ] Asset path is referenced from `brand-memory.json`.
- [ ] Colors match approved tokens.
- [ ] Typography follows approved rules.
- [ ] Usage constraints are documented.

## Web/UI

- [ ] SVG opens correctly in browser.
- [ ] PNG has transparent background when needed.
- [ ] Logo works on light and dark backgrounds.
- [ ] Contrast is acceptable.
- [ ] Asset remains legible at small sizes.

## Print

- [ ] Resolution is sufficient for target size.
- [ ] Color mode requirements are known.
- [ ] Safe area and bleed are defined when needed.
- [ ] Fonts are outlined or packaged when required.

## Cricut/Vinyl

- [ ] SVG uses flat vector paths.
- [ ] Text is converted to outlines.
- [ ] Strokes are expanded.
- [ ] No pattern fills.
- [ ] No gradients in cut SVG.
- [ ] No embedded raster images in cut SVG.
- [ ] Layers are separated by vinyl color when needed.
- [ ] Tiny details are simplified for cutting.
- [ ] Upload tested in Cricut Design Space when possible.
