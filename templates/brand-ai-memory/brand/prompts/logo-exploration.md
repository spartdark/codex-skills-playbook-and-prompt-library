# Logo Exploration Prompts

## Visual Territories

```md
Actua como director de identidad visual.

Contexto:
- Marca:
- Audiencia:
- Personalidad:
- Valores:
- Usos principales:
- Restricciones:

Genera 5 territorios visuales distintos.
Para cada territorio incluye:
- concepto;
- tipo de logo recomendado;
- paleta tentativa;
- tipografia sugerida;
- estilo de imagen;
- riesgos;
- prompt para generar moodboard o logo mark.

Marca supuestos y no tomes decisiones finales.
```

## Image Model Prompt

```md
[brand name] [logo type] for [industry/category], [personality traits], [visual style], clean vector-style mark, simple shapes, strong silhouette, works in one color, no text, no mockup, no 3d, no gradients
```
