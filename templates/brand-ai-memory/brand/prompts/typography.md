# Typography Prompts

## Typography Direction

```md
Actua como especialista en identidad tipografica.

Usa este contexto:
- Marca:
- Audiencia:
- Personalidad:
- Logo direction:
- Use cases:
- Constraints:

Propone 3 sistemas tipograficos.
Para cada uno incluye:
- headline font;
- body font;
- fallback;
- razon de uso;
- riesgos de legibilidad;
- compatibilidad con web/print;
- si conviene para wordmark o solo para texto de soporte.
```

## Review Prompt

```md
Evalua esta propuesta tipografica contra `brand-memory.json`.

Revisa:
- consistencia con personalidad;
- legibilidad;
- disponibilidad/licencia;
- web fallback;
- uso en corte o vinilo;
- riesgos.

No apruebes fuentes sin marcar dependencias o licencias pendientes.
```
