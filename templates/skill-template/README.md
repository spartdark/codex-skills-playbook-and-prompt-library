# Template Skill

Esta plantilla ayuda a crear nuevas skills públicas dentro de esta librería sin empezar desde cero.

## Incluye

- `SKILL.md` breve y operativo;
- `prompt.md` con estructura sugerida;
- `skill.json` con metadata instalable;
- ejemplo de uso;
- asset reutilizable como checklist editorial.

## Cómo usarla

1. copia esta carpeta;
2. renombra el slug;
3. ajusta `skill.json`;
4. reescribe `SKILL.md` y `prompt.md`;
5. agrega un ejemplo real.
