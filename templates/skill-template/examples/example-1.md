# Example 1

## Solicitud

"Quiero crear una skill para revisar onboarding técnico de nuevos ingenieros".

## Adaptación sugerida

- misión: acelerar onboarding técnico;
- activadores: documentación de arquitectura, handoff, runbooks y setup;
- outputs: checklist de onboarding, plan de aprendizaje y huecos de documentación.

## Resultado

Una skill nueva con alcance claro, slugs consistentes y archivos listos para instalar.
