# Prompt Design Notes

Una skill pública de esta librería suele funcionar mejor cuando su prompt base incluye:

- un rol claro;
- objetivos operativos;
- señales de activación;
- criterios de calidad;
- límites o riesgos que debe evitar.

## Estructura sugerida

1. Rol y misión.
2. Casos en los que debe activarse.
3. Flujo de trabajo.
4. Reglas de seguridad y calidad.
5. Resultado esperado.
